Quick Reference

API Endpoints

Operation SOAP Action Purpose
Register Invoice urn:VeriFactu/Alta Register new invoices with AEAT
Cancel Invoice urn:VeriFactu/Baja Cancel previously registered invoices
Query Invoice urn:VeriFactu/Consulta Query invoice status and details

Required Fields

Header Information (Cabecera)

Field Type Required Description
NIF String Yes Tax identification number
Fecha Date Yes Invoice date (YYYY-MM-DD)
Hora Time Yes Invoice time (HH:MM:SS)
Serie String No Invoice series
NumFactura String Yes Invoice number

Invoice Data (Factura)

Field Type Required Description
ImporteTotal Decimal Yes Total amount without taxes
ImporteTotalConImpuestos Decimal Yes Total amount including taxes
TipoFactura String Yes Invoice type (F1, F2, etc.)
ClaveRegimenEspecialOTrascendencia String Yes Special regime key

Response Codes

Code Description Action
0 Success Continue processing
1001 Invalid NIF format Check NIF format
1002 Invalid invoice number Check invoice number format
1003 Invalid amount format Check amount format
2001 Service temporarily unavailable Retry later
2002 Database connection error Contact support
2003 Authentication failed Check certificate

Certificate Requirements

Client Certificate

  • X.509 format with private key
  • Obtained from FNMT
  • Valid for 2-3 years
  • Stored securely on server

Cost

  • Individual (Persona Física): FREE
  • Company (Persona Jurídica): €14-20

Database Schema

-- Add these columns to your invoices table ALTER TABLE invoices ADD COLUMN aeat_csv_code VARCHAR(100); ALTER TABLE invoices ADD COLUMN aeat_status VARCHAR(20); ALTER TABLE invoices ADD COLUMN aeat_error TEXT;

Column Descriptions

  • aeat_csv_code: Secure verification code returned by AEAT
  • aeat_status: Invoice status (verified, error, pending)
  • aeat_error: Error message if registration fails

Common Error Scenarios

Network Errors

  • Connection timeout
  • Certificate validation failed
  • DNS resolution failed

SOAP Errors

  • Invalid XML format
  • Missing required fields
  • Authentication failed

Business Logic Errors

  • Invalid invoice data
  • Duplicate invoice number
  • Tax calculation errors
