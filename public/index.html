Find out all APIs at API Reference section.

Authentication

When working with the API, you need to use mutual TLS authentication with a qualified electronic certificate.

Certificate Requirements

  • Format: X.509 certificate with private key
  • Authority: FNMT (Fábrica Nacional de Moneda y Timbre)
  • Storage: Must be securely stored on your server
  • Validity: Typically 2-3 years, must be renewed before expiration

Obtaining the Certificate

To get your certificate, visit the FNMT website and complete the registration process:

  • For Individuals (Persona Física): FREE
  • For Companies (Persona Jurídica): €14-20
