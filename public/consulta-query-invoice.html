Query Invoice (Consulta)

Overview

The Consulta operation is used to query the status and details of previously registered invoices with the Spanish tax authority (AEAT). This operation allows you to verify invoice status and retrieve verification codes.

Purpose

  • Status Verification: Check invoice registration status
  • Data Retrieval: Retrieve invoice details and CSV codes
  • Audit Trail: Maintain compliance records
  • Error Resolution: Troubleshoot registration issues

SOAP Action

SOAPAction: "urn:VeriFactu/Consulta"

Request Structure

SOAP Envelope

Required Fields

Header Information (Cabecera)

  • NIF: Tax identification number of the issuer
  • Fecha: Date of the invoice
  • Hora: Time of the invoice
  • Serie: Invoice series (optional)
  • NumFactura: Invoice number to query

Invoice Data (Factura)

  • ImporteTotal: Total amount of the invoice
  • ImporteTotalConImpuestos: Total amount including taxes
  • TipoFactura: Type of invoice (F1, F2, etc.)
  • ClaveRegimenEspecialOTrascendencia: Special regime key

Response Structure

Success Response

0 OK ABC123456789 REGISTRADA 2024-01-15T10:30:00
Success Response Fields:
  • Codigo: Response code (0 = success)
  • Descripcion: Description of the result
  • CSV: Secure verification code for the invoice
  • Estado: Current status of the invoice
  • FechaRegistro: Registration date and time

Invoice Status Values

REGISTRADA

Invoice successfully registered with AEAT

ANULADA

Invoice has been cancelled/voided

PENDIENTE

Invoice registration is pending

ERROR

Invoice registration failed with errors

Error Handling

Common Error Codes

Query Errors

  • 5001: Invoice not found
  • 5002: Invalid query parameters
  • 5003: Access denied

System Errors

  • 6001: Service temporarily unavailable
  • 6002: Database connection error
  • 6003: Authentication failed
